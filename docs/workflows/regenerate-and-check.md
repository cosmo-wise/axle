# Regenerate and check

Use this loop after any descriptor edit:

```bash
go run ./cmd/axle gen --descriptor <descriptor> --out <generated-dir> --json
go run ./cmd/axle gen --descriptor <descriptor> --out <generated-dir> --check --json
go run ./cmd/axle check --descriptor <descriptor> --root . --json
go test ./...
```

Generated files carry `Code generated by axle gen; DO NOT EDIT.` Do not manually patch generated files; change the descriptor and regenerate.
