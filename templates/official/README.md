# Official Axle Templates

Axle code generation templates are compiled into Go source code in `internal/cli/app_templates.go`.
Each template renders a parsed `ResourceDescriptor` as its data context and produces:

- Go source files (types, routes, registry)
- OpenAPI specs
- SQL migration files
- App scaffold files (README, main.go, app.go, verify script)

Templates use Go's `text/template` format and are tested through `testdata/examples/generated-backend`
and unit tests in `internal/cli/`.

Generated files carry `Code generated by axle gen; DO NOT EDIT.` — change the descriptor
or catalog manifest and regenerate instead of editing output directly.
